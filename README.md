## About CMake CLI Tools

Purpose: This is a project with a set scripts to help automate the creation of CMake projects for C/C++ software
development. Also these tools are an attempt to help ease the handling the project as it grows in size and evolves.

## Getting Started

### Prerequisites

Dependencies need to be installed before being about to use the tools locally in your computer.

### Installation

To install the bash versions of the tools follow the steps below

1. clone repo to your machine
   ```sh
   git clone https://github.com/fonzg1/CMake_cli_tools.git
   ```
2. run the following script to add tools to your PATH
   ```
   bash setup_scripts2path
   ```
